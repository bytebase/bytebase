Design Specification

Key Changes:

1. Colored sidebar bar via Slack attachments.color — instant visual status
2. Issue title as a clickable link — the primary CTA, not buried in metadata
3. Compact metadata line — Project + Creator/Environment on one line with | separator
4. Removed noise — No Project ID, no redundant Actor field
5. Description as plain text — not wrapped in code block
6. Bytebase footer — clean branding like GitHub's footer
Color Mapping
INFO (created)
WARN (approval, sent back)
SUCCESS (approved, completed)
ERROR (failed)
Slack Block Kit Structure
attachments[0].color = sidebar
attachments[0].blocks = content
Top-level blocks = button
Removed Fields
Project ID (technical noise)
Actor line (redundant with description)
Added Elements
Linked issue/rollout title
Compact metadata row
Bytebase footer
